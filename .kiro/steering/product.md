# Product Overview

Lyricsify is a Spotify lyrics viewer designed for multi-taskers. The application provides an efficient way to view song lyrics while working or performing other tasks.

## Target Use Case

- Display real-time lyrics for currently playing Spotify tracks
- Optimized for users who want lyrics visible while multitasking
- Lightweight and non-intrusive interface
